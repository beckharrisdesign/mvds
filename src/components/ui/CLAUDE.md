# src/components/ui — vendored shadcn/ui

These files are **generated by the shadcn CLI** and must stay updatable. Do **not**
hand-edit them.

- Add a component: `npx shadcn@latest add <name>` (run from repo root).
- Need a tweak? Prefer wrapping/composing in your own component, or adjust the
  **token layer** (`src/index.css`) so the change flows through every component.
- Theming is via CSS variables + the 8-grid spacing system — see [AGENTS.md](../../../AGENTS.md).
